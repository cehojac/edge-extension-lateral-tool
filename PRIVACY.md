# Privacy Policy - Edge Sidebar Productivity Extension

**Last Updated:** June 11, 2026

## Overview

This privacy policy explains how the Edge Sidebar Productivity Extension ("the Extension") handles your data. The Extension is designed with privacy in mind and does not collect, store, or transmit any personal data to external servers.

## Data Collection

### No Personal Data Collection
The Extension does **NOT** collect any personal information, including:
- No names, email addresses, or contact information
- No browsing history or user behavior tracking
- No location data or device identifiers
- No usage analytics or performance monitoring
- No third-party tracking or advertising

### Local Data Storage Only
All data generated by the Extension is stored **locally** on your device using the browser's `chrome.storage.local` API. This data never leaves your device unless you explicitly choose to export it.

## Data Stored Locally

The Extension stores the following data locally on your device:

### 1. Tool Configuration
- Tool order and visibility preferences
- Theme preferences (light/dark/system)
- Custom tool settings

### 2. User-Generated Content
- **Todo List**: Tasks you create, their completion status
- **Quick Notes**: Notes you write in the notepad
- **Shortcuts**: Website names and URLs you add
- **Calculator**: No data stored (temporary calculations only)
- **Timer**: No data stored (temporary timers only)

### 3. Translation Data
- **Translation Models**: The Extension uses Microsoft Edge's built-in Translator API
- Translation models are downloaded locally to your device the first time you use a language pair
- These models are provided by Microsoft and are stored in your browser's cache
- No translation text is sent to external servers (except during initial model download by Microsoft Edge)

## Data Retention

- Data is stored indefinitely on your device until you choose to delete it
- You can delete all data at any time through the Extension's settings
- Uninstalling the Extension will remove all stored data from your device
- Data is not synchronized across devices (unless you use browser sync, which is managed by your browser, not this Extension)

## Data Sharing

### No Data Sharing
The Extension does **NOT** share any data with third parties, including:
- No data is sold to advertisers or data brokers
- No data is shared with analytics companies
- No data is transmitted to external servers (except as noted below for Microsoft Edge Translator API)

### Microsoft Edge Translator API
- The Extension uses Microsoft Edge's built-in Translator API
- Translation models are downloaded from Microsoft's servers the first time you use a language pair
- After download, all translation happens locally on your device
- No translation text is sent to Microsoft servers after the initial model download
- This is the same API used by Microsoft Edge's built-in translator

## Data Security

- All data is stored using the browser's secure `chrome.storage.local` API
- Data is encrypted at rest by your browser's security mechanisms
- The Extension does not have access to data from other websites or your browsing history
- The Extension cannot access data from other extensions

## Your Rights

### Access and Control
You have the right to:
- **View** all data stored by the Extension through your browser's developer tools
- **Delete** all data at any time through the Extension's settings
- **Export** your data by copying it manually from the browser's storage
- **Uninstall** the Extension to remove all data from your device

### How to Delete Your Data
1. Open the Extension settings
2. Click "Clear all data"
3. Confirm the action
4. All stored data will be permanently deleted from your device

## Third-Party Services

### No Third-Party Services
The Extension does not use any third-party services for:
- Analytics
- Advertising
- Data processing
- Cloud storage

### Microsoft Edge API
The only external service used is Microsoft Edge's built-in Translator API, which is:
- Provided by Microsoft as part of the browser
- Used only for downloading translation models
- Does not transmit your translation text after initial download
- Governed by Microsoft's privacy policy

## Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect any information from children.
